# Trabalho - SGBD Orientado a Objetos

## Banco Escolhido

ObjectDB

## Aluna

Mariana Araújo de Medeiros

## Disciplina

Projeto e Administração de Banco de Dados

## Descrição

Este projeto apresenta a utilização do ObjectDB, um Sistema Gerenciador de Banco de Dados Orientado a Objetos (SGBDOO), demonstrando sua instalação, configuração e acesso através de uma aplicação Java.

## Aplicação Desenvolvida

Foi criada uma aplicação Java simples que realiza a conexão com o ObjectDB, insere um objeto da classe Pessoa e posteriormente consulta os registros armazenados.
